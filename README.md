# Gradle-Maven Test
This is an example of a web application as both a Gradle and Maven Project.
Both builds produce the same result.

Tools used:
- Java 11
- Spring
- Spring Boot 2.4.3
- Spring MVC
- JPA
- Hibernate 5
- JUnit
- Maven 4.0.0
- Gradle 6.8.3
- H2 Database
- HTML5
- Bootstrap 4.0.0
- Thymeleaf
